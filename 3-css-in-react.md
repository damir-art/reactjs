# CSS в React

Что бы использовать CSS в React, добавьте теги style в head или body:

    <style>
        #root {
            background-color: #eee;
        }
        #root h1 {
            color: #333;
            font-size: 56px;
            margin: 0;
            padding: 10px 20px;            
        }
    </style>
